## 🧠 Что такое DNS
Система, которая переводит имена сайтов (`google.com`) в IP-адреса (`142.250.185.46`).  
«Телефонная книга интернета».

---

## 📚 Основные типы записей

| Тип | Назначение | Пример | Простыми словами |
|-----|-----------|--------|------------------|
| **A** | Домен → IPv4 | `site.com → 93.184.216.34` | Прямой адрес сервера |
| **AAAA** | Домен → IPv6 | `site.com → 2001:db8::1` | Адрес для нового интернета |
| **CNAME** | Домен → Другой домен | `www.site.com → site.com` | Псевдоним, указатель |
| **MX** | Почтовый сервер | `site.com → mail.site.com` | Куда отправлять письма |
| **TXT** | Текстовая информация | `site.com → "v=spf1 ..."` | Для проверки почты, домена |
| **NS** | DNS-сервер для зоны | `site.com → ns1.site.com` | Кто отвечает за домен |

---

## 🔄 Как это работает (по шагам)

1. Ты вводишь `brkserv.duckdns.org` в браузере.
2. Компьютер спрашивает DNS-резолвер (например, `8.8.8.8`).
3. Резолвер ищет:
   - Идёт к корневому серверу → узнаёт, где `.org`
   - Идёт к серверу `.org` → узнаёт, где `duckdns.org`
   - Идёт к серверу `duckdns.org` → спрашивает про `brkserv`
   - Получает **A-запись**: `147.45.173.98`
4. Резолвер возвращает IP-адрес браузеру.
5. Браузер соединяется с `147.45.173.98:443`.

## 📋 Популярные DNS-серверы

| Адрес | Кто | Особенность |
|-------|-----|-------------|
| `8.8.8.8` | Google | Быстрый, популярный |
| `1.1.1.1` | Cloudflare | Приватный, быстрый |
| `9.9.9.9` | Quad9 | Безопасный (блокирует плохие сайты) |
| `77.88.8.8` | Яндекс | Работает в РФ |

---

## 🛠️ Полезные команды

```bash
# Узнать A-запись
dig brkserv.duckdns.org A

# Узнать CNAME
dig s-ui.brkserv.duckdns.org CNAME

# Узнать NS-серверы для домена
dig duckdns.org NS

# Кто отвечает за домен (трассировка)
dig +trace brkserv.duckdns.org

# Прямой запрос к конкретному DNS-серверу
nslookup brkserv.duckdns.org 1.1.1.1
```

## ✅ Закрепление

| Ты хочешь | Что создать |
|-----------|-------------|
| Чтобы `мой-сайт.duckdns.org` показывал твой сервер | A-запись → твой IP |
| Чтобы `панель.мой-сайт.duckdns.org` вело туда же | CNAME → `мой-сайт.duckdns.org` |
| Чтобы почта шла на Яндекс | MX-запись → `mx.yandex.ru` |
| Чтобы Google знал, что сайт твой | TXT-запись с верификационным кодом |

---

**Запомни одной фразой:**  
*A-запись — это «вот мой адрес». CNAME — это «спроси у того».*
