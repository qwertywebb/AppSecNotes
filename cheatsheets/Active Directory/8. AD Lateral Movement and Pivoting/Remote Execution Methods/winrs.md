# Отчёт: Аутентификация через WinRM с PtH/PtK

# WinRS — это клиент (команда winrs.exe), который подключается к службе WinRM на удаленной машине.
## Описание

После внедрения хэша или ключа через Mimikatz(overpass-the-hash), WinRM может использоваться для подключения к удалённой машине без указания учётных данных.

## Подключение через winrs

```cmd
winrs.exe -r:TARGET.za.tryhackme.com cmd
```

WinRS автоматически использует креды из текущей сессии (уже внедрённые через Mimikatz).
