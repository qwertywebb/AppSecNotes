# VLAN (Virtual Local Area Network) — это технология, которая позволяет на одном физическом коммутаторе создать несколько независимых сетей.

# Представь: У тебя есть один коммутатор (коробка с портами). Ты говоришь: "Порты 1-5 будут сетью для Бухгалтерии (VLAN 10)". И "Порты 6-10 будут сетью для Отдела кадров (VLAN 20)".

# Компьютеры из Бухгалтерии не увидят компьютеры из Отдела кадров, и наоборот, хотя физически они подключены к одному ящику.

# Чтобы интернет был и там, и там, коммутатор соединяется с роутером одним проводом. Но этот провод — как "трубопровод". По нему идут данные из обеих сетей. Чтобы роутер не запутался, на пакеты данных клеят цветные стикеры (теги): синий для VLAN 10, красный для VLAN 20.

# Роутер, в свою очередь, работает с сабинтерфейсами и настроен так, что знает: "пакеты с синим стикером отправляю в сеть 192.168.1.0, а с красным — в сеть 192.168.2.0".
